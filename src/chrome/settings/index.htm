Access to Gmail is limited only to this single computer. learn more

To ensure the smoothest experience when using CryptUP, it needs to connect to the Gmail API. This allows your local CryptUP plugin to use Gmail functions to encrypt and decrypt messages when you choose so.

When you connect to Gmail, the plugin will receive API Tokens. These tokens are needed to access your Gmail inbox programmatically, and they are only saved on your computer, locally. They will never leave your own browser, and Chrome will destroy them when your uninstall CryptUP.

In other words, these Tokens are handled so securely, that even CryptUP developers cannot and will not be able to access them.

Because the code running CryptUP is open-source, anyone can verify how these Tokens are handled. To learn more, or inspect the code, visit CryptUP Sources Repository on GitHub. Alternatively, you are welcome to ask me directly at tom@cryptup.org
